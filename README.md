## ✨ 功能特点

- **智能识别**：使用连通域分析自动检测每个独立贴纸，无需手动设置网格
- **精准抠图**：严格的绿幕检测算法，只移除纯绿色背景，不误伤角色
- **透明动图**：生成真正的 GIF 动图格式，完美支持微信透明表情
- **一键操作**：拖拽图片到 BAT 文件即可自动处理

## 📦 文件说明

| 文件 | 说明 |
|------|------|
| `sticker_splitter.py` | Python 核心脚本 |
| `拖入图片自动分割.bat` | 一键处理脚本（拖入图片即可运行） |
| `表情包.md` | AI 生成提示词（皮克斯级萌感版） |

## 🚀 使用方法

### 方法一：拖拽运行（推荐）
1. 将你的绿幕贴纸图片拖到 `拖入图片自动分割.bat` 上
2. 等待处理完成
3. 在 `output_stickers` 文件夹中获取透明 GIF 表情

### 方法二：命令行运行
```bash
# 基础用法
python sticker_splitter.py 你的图片.png

# 指定输出目录
python sticker_splitter.py 你的图片.png -o 我的表情包

# 限制最长边尺寸
python sticker_splitter.py 你的图片.png -m 200
```

## 📱 微信表情添加方法

1. 在微信聊天界面点击 **【表情图标】**
2. 点击 **【自定义表情（爱心图标）】**
3. 点击 **【+号】** 添加表情
4. 从相册选择生成的 GIF 文件

> ⚠️ **注意**：不要直接在聊天框发送图片，否则透明背景会丢失！
