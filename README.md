## 🚀 使用方法

### 方法一：拖拽运行（推荐）
1. 将你的绿幕贴纸图片拖到 `拖入图片自动分割.bat` 上
2. 等待处理完成
3. 在 `output_stickers` 文件夹中获取透明 GIF 表情

### 🟢 微信发送说明
**现在支持直接发送！**
由于生成的是真·透明 GIF 格式，你可以直接将 `output_stickers` 里的文件拖入微信聊天窗口，透明背景**完美保留**，不会变白底！

### 方法二：命令行运行
```bash
# 基础用法
python sticker_splitter.py 你的图片.png

# 指定输出目录
python sticker_splitter.py 你的图片.png -o 我的表情包
```

## 📱 微信表情添加方法

1. 在微信聊天界面点击 **【表情图标】**
2. 点击 **【自定义表情（爱心图标）】**
3. 点击 **【+号】** 添加表情
4. 从相册选择生成的 GIF 文件

> ⚠️ **注意**：不要直接在聊天框发送图片，否则透明背景会丢失！
